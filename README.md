# Echonade w/ ACP

## Quick Start

1. Install dependencies:
```bash
pnpm install
```

2. Run the application:

### Agentic Mode
```bash
pnpm dev:agentic
```

### Reactive Mode
```bash
pnpm dev:reactive
```

## Project Structure

- `src/` - Source code directory
  - `agentic/` - Agentic implementation
  - `reactive/` - Reactive implementation
